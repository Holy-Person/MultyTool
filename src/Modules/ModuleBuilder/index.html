1: Getting Started
Introduction
Welcome to the tutorial, here we're going to start off with the first steps toward your own projects.
Creating your first Module
Everything has to start somewhere and this is where our first module is going to be born.
Your first step should be to obtain a usable module with a template. Open the "Modules" folder, copy the "ExampleModule" and give it the name you want to give your module.
You should now have a folder with 4 files in it, the HTML is your module's main content, the CSS is for styling and the JS is for interactions.

Now that you have your module folder we can start with setting your module up. Right now you're probably unable to open your module from the menu, thats ok. Open your module folder and head to the file named "moduleInfo.json", then change the module's name to the folder name. Save the edits, refresh the main menu and you should now be able to open your module.
If you want to learn more about editing the moduleInfo I suggest you head over to the ToolByte specific help section.
Editing the Module's Content
Now that we have our first module created we can begin with actually putting our own contents into it. There are 3 different things that work together to form a module, HTML, CSS and JavaScript.
You can skip these parts if you know how to code already.
For starters let's change the HTML a bit, this is the main visible content of your module, head over to your "index.html" and look for the "body" tag, this is where all the visible stuff is.
There should be some comments on the right side of your code, they should explain a few more things on what to do.
Due to this just being a test so far from here on your on your own.
